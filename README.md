# Cameos

[![Build Status](https://travis-ci.org/idxp/cameos.svg?branch=master)](https://travis-ci.org/idxp/cameos)

The IDXP Analytics collection of Ansible roles. We have roles for:

* supervisor
* beanstalkd
* ruby & rbenv
* python & miniconda
* python & pyenv
* nginx
* postgresql
* consul
* memcached
